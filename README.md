# Film Finder

Film-Finder é um web-app para os amantes de cinema que querem informações dos seu filmes favoritos. Projeto feito em JavaScript utilizando a API da OMDB para consulta dos filmes.

---

## Technologies

- JavaScript (ES6)
- HTML
- CSS
- Bootstrap 4
